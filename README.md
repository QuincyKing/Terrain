### 简介
这个软件主要通过对广义三棱柱(GTP)进行修正与扩展,面向覆岩变形参数的需要, 提出了基于关键层理论覆岩层弯曲带三维地质数据模型。并且模拟出采动后覆岩层弯曲带下沉移动的结果，对预估开采结果具有重要意义。
